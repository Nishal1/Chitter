# Chitter

## Installation
1. Clone the repo.
2. run 'npm install'

## Run app

You can run this application on localhost.

### Locally on Localhost

port: 3000 `npm run start:dev`

## Available Scripts:
1. "start": "node app.js",
2. "start:dev": "nodemon app.js",
3. "prettier:test": "prettier --check \"**/**/*.js\"",
4. "prettier:fix": "prettier --write \"**/**/*.js\""
